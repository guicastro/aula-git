Comandos GIT

- CONFIGURAÇÃO INICIAL
git config --global user.name
git config --global user.email

- CRIAR REPOSITÓRIO
1) cessar o diretório onde está o repositório
2) git init

- ADICIONAR ARQUIVO
git add

- REALIZAR O COMMIT
git commit -m "Mensagem do log do commit"

- REMOVER ARQUIVO DO COMMIT E DEIXAR COMO UNSTAGE
git reset HEAD <nome do arquivo>

- VER O LOG
git log -p

- VER AS ESTATÍSTICAS DO LOG FORMATADO
git log --pretty=format:"%h - %an, %ar : %s"

- VER O LOG DE COMMITS DOS ÚLTIMOS 2 DIAS
git log --since=2.days

- VOLTAR COMMIT
git checkout hash_do_commit_a_restaurar

- VOLTAR COMMIT ATÉ O PONTO ONDE O ARQUIVO FOI ADICIONADO (o ˜1 indica o último commit, pode ser 2, 3, etc)
git reset HEAD˜1 --soft
(trocando o --soft por --hard remove todos os commits)


- VOLTAR COMMIT (o ˜1 indica o último commit, pode ser 2, 3, etc)
git reset HEAD˜1

- VOLTAR O ARQUIVO A VERSÃO ORIGINAL
git checkout -- nome_do_arquivo.php

- MOSTRAR A BRANCH ATUAL
git branch

- CRIAR BRANCH
git checkout -b 0.1.0

- PARA JUNTAR BRANCHES (MERGE)
1) acessar a branch que vai receber a outra branch
2) git merge nome_da_outra_branch
3) Preencher a mensagem de Commit e depois :wq

- PARA JUNTAR BRANCH NÁO MOSTRANDO COMMIT (REBASE)
1) acessar a branch que vai receber a outra branch
2) git rebase nome_da_outra_branch

- PARA REMOVER UMA BRANCH
git branch -D nome-do-branch

- PARA CONFIGURAR UM REPOSITÓRIO REMOTO
git remote add origin https://github.com/guicastro/aula-git.git

- PARA CRIAR UMA BRANCH NO REPOSITÓRIO REMOTO
git push origin master

- PARA ENVIAR UM REPOSITORIO (PUSH)
git push nome_repositorio_remoto nome_branch_local
(o padrão de nome_repositorio_remoto é origin)

- PARA CLONAR UM REPOSITORIO REMOTO
git clone endereco_do_repositorio_remoto nome_pasta_local
(nome da pasta local somente se desejar, caso não definido usará o nome do repositório remoto)

- PARA CRIAR UMA BRANCH LOCAL BASEADO EM UMA BRANCH DE UM REPOSITÓRIO REMOTO
git checkout -b nome_branch_local caminho_branch_remoto
(para ver as banches remotas, usar git branch -a)

- PARA OBTER AS ALTERAÇÕES DO REPOSITÓRIO REMOTO
git pull nome_repositorio_remoto nome_branch
